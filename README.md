# Utility Tools

A lightweight day-to-day utility web app with common tools for all users.

## Features
- Basic calculator (`+`, `-`, `*`, `/`)
- Kilometer to miles converter
- Age calculator (from date of birth)
- Random password generator + copy
- Quick notes with local storage
- QR code generator
- PDF utility hub:
  - Text to PDF
  - Images to PDF
  - Word/TXT to PDF
  - PDF to Word (.doc)
- Multi-theme UI switcher (Corporate, Midnight, Emerald, Sunset)
- Pass / PIN generator
- Case converter
- Encoder / decoder (Base64 + URL)
- Diff checker
- Regex tester
- Cron generator
- Budget box (income/expense tracker)
- Dashboard tool search
- Global clock (current + world clocks in one tool)
- All-country support in global clock (auto-loaded country list)
- Live currency converter with fluctuation trend and auto-refresh
- Language support (English/Hindi)

## Run Locally
From project folder:

```bash
python3 -m http.server 8000
```

Then open:
`http://localhost:8000`

## Files
- `index.html` - UI structure
- `styles.css` - styling and responsive layout
- `app.js` - all tool logic
